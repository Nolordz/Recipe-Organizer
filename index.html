On this project we can see how we can manipulate data using python to create models, but first let me explain what is Django. DJANGO : is a web application frame work,written in python language, that can be use to create complex data models(websites) in a more easy and friendly way, (Some examples that use django are websites like Mozilla and instagram). Django has what is call "MVC" framework , "Model View Controller" that uses relation between data models(Known as classes in python) and the database(Models), has also a system that handles and process request (View) with a web templating system, and a URL as a dispatcher (Controller).
